# Challenge Brief

## Objective

Build a production-ready Superteam Australia website that reflects global Superteam standards with a distinct Australian ecosystem identity.

## Required product surface

- Landing page with hero, mission, impact stats, events, members showcase, partners, community proof, FAQ, and CTA.
- Members directory page with search/filter.
- Admin-manageable CMS content (no code-only edits).

## Technical expectations

- Next.js + React + Tailwind CSS
- Responsive UX (desktop + mobile)
- Supabase as preferred backend/CMS
- Luma events integration
- Performance and SEO readiness

## CMS requirements

Admins should manage:

- Events
- Members and featured spotlights
- Partners
- Landing page content sections

Recommended: role-based access controls and practical authoring UX.

## Deliverables

- Figma designs + prototype
- Clean production-grade repository
- Setup and architecture documentation
- Optional live deployment
